Hotelia est un mini ERP de gestion hôtelière moderne, conçu pour gérer efficacement les opérations d’un hôtel : clients, chambres, réservations et facturation.

Ce projet évolue d’une architecture monolithique Java EE vers une architecture moderne basée sur une API REST, un frontend découplé et une infrastructure conteneurisée.

### Stack technique

- Backend
  Spring Boot
  Spring Data JPA
  MySQL
- Frontend
  React
  Axios (communication API)
  React Router
- Infrastructure
  Docker
  Docker Compose
